-------------------------------------------------------------------------------
-- CorsixTH Graphics Project
-------------------------------------------------------------------------------

This project aims to replace the original graphics from Theme Hospital that
are currently used in CorsixTH. So far the following artists have made
contributions:

Zephyris

-------------------------------------------------------------------------------

How to use the sprites in this repository:

1. Compile the AnimationEncoder

2. Compile each animation file

3. Put each animation file in a Graphics directory of your choice, for example
   in Windows the one where you installed CorsixTH/Graphics

4. Instruct CorsixTH to use new graphics by enabling it in the config file.

5. Enjoy the game!
